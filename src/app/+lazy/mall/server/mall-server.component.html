售后问题

  • 质量保障

    云深间造为出售的产品提供为期一年的修理、更换、退货（“三包”）保障。“三包”保障的起始日以产品实际签收日期为准，请您在签收交货单据时，注意查看交货日期。“三包”质保期内,在正常使用和维护的情况下,经检验属于产品材料和工艺问题而导致的损坏，本公司将免费维修，直至更换。保障范围：

    修理：在“三包”期限内，产品质量问题属于修理范围；

    更换：在“三包”期限内，同一故障经两次维修后仍未能达到质量标准的进行更换；

    退货：在“三包”期限内， 同一故障经两次更换后仍无法达到质量标准的，您可以选择更换其他同类产品或者退货。

  • 维修服务

    三包”有效期内，云深间造将提供优质服务并负责相关费用(包括材料、工时等费用，双方另有约定的情况除外)；“三包”有效期外，云深间造将根据实际情况收取原材料成本及上门维修费，具体费用将会在服务发生前与您商定。

    不属于“三包”范围的特例情况：

    没有按照云深间造接货标准程序操作的情况拆包引起的损坏；

    没有按照组装说明书要求正确操作引起的产品损坏；

    没有按照组装说明书要求，自行更换部件引起的损坏；

    客户自行拆装,调换产品到错误的使用环境引起的损坏；

    不正确的维护保养、或者没有按照云深间造产品使用和保养手册操作引起的损坏；

    非专业人士修理引起的损坏；

    由于不正当使用产品和产品部件而造成的人身和财产损失；

    客户不能提供购买发票与保修卡,或者擅自涂改保修卡；

    已经超过云深间造退换货和维修期限的产品；

    因其它不可抗拒力(地震,火灾)等造成的产品损坏；

    其它使用、维护、保管不当及人为损坏的情况；

    玻璃镜子发生霉变和雾光的三包期限为六个月，处理家具商品只享受包修服务。
